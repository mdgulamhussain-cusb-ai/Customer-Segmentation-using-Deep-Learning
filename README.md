# Customer Segmentation using Deep Learning

An end-to-end unsupervised learning project that segments customers into meaningful behavioral groups using deep learning.

## Tech Stack
- TensorFlow / Keras
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Features
- Autoencoder-based feature learning
- KMeans clustering for segmentation
- PCA-based visualization dashboards
- Modular, production-ready codebase

## How to Run
```bash
pip install -r requirements.txt
python main.py
